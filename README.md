# ⚔️ Slayken

**Slayken** ist ein mobiles Anime-RPG-Tap-Game mit Gacha-Mechaniken, coolen Effekten und deinem einzigartigen Style.

---

## 🚀 Features

- 🎮 Tap-to-Fight Gameplay
- 🎴 Charakter-Beschwörung (Gacha-System)
- 🌌 Rotierende animierte Hintergründe
- 📈 EXP-, Coin- & Crystal-Events
- 👥 Team-Auswahl aus beschworenen Charakteren
- ⚙️ Einstellungsseite mit Reset-Funktion
- 🎵 Eigene Musik & Soundeffekte
- 📦 Offline-Speicherung via `AsyncStorage`

---

## 🛠️ Tech Stack

- [React Native](https://reactnative.dev/)
- [Expo](https://expo.dev/)
- [Lottie Animations](https://airbnb.io/lottie/)
- [AsyncStorage](https://github.com/react-native-async-storage/async-storage)
- [ESLint + Prettier](https://eslint.org/ & https://prettier.io/)
